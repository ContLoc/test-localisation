Managing users groups
=================================

This section describes actions with Platform users groups.

To go to the users group management page, click 'Tenant Configuration' → 'Groups':

.. thumbnail:: ../\_static/media/config\_menu\_groups.png

It will open the users group management page will open:

.. thumbnail:: ../\_static/media/user\_groups\_common\_1.png

1. Search box to find a group by name.
2. Page refresh button.
3. Groups can be filtered by type: local or external.
4. The 'Add External Group' button opens the :ref:`'External authentication' <ldap>` page.
5. The 'Add Local Group' button opens a window to add a new group.
6. Gear icon, using which you can fix, delete or change the positions of columns in the table.

	.. thumbnail:: ../\_static/media/group\_table\_setting.png :width: 300 :align: center

7. Information on groups:

	* group name;
	* group description;
	* group type: local or external;
	* number of users added to the group;
	* number of workspaces that users in the group have access to.

8. Group menu - allows you to delete the group. In the modal window, enter 'delete' and click 'Delete Group'.

	.. thumbnail:: ../\_static/media/user\_groups\_common\_1.png :width: 300 :align: center

.. note::

	To filter information in the table, click the header of the column.

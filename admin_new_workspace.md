Adding a new workspace
=======================================

	.. attention::

		Only System Owners and Tenant Owners can add a new workspace.

		To add PostgreSQL  :ref:`instances <instance_adding>` to a workspace, you need to create a workspace first.

To add a new workspace:

\#. Go to workspaces page and click 'Add workspace'.

	.. thumbnail:: ../_static/tenants/image42.png

\#. In the opened window, enter the name of the new workspace in the text field. 'Workspace name' field is mandatory. The name shall not duplicate other names of workspaces.

	.. thumbnail:: ../_static/tenants/image118.png
		:width: 300
		:align: center

\#. Click 'Add'.

Добавление нового рабочего пространства
=======================================

	.. внимание::

		Добавлять новое рабочее пространство могут только владельцы систем и владельцы-арендаторы.

		Чтобы добавить PostgreSQL:ref:`экземпляры <instance_addding>` в рабочую область, сначала нужно создать рабочую область.

Добавление нового рабочего пространства:

\#. Перейдите на страницу рабочих областей и нажмите кнопку 'Добавить рабочую область'.

	.. thumbnail:: ../_static/tenants/image42.png

\#. В открывшемся окне введите в текстовое поле имя новой рабочей области. Поле 'Имя рабочей области' обязательно. Название не должно дублировать другие названия рабочих пространств.

	.. thumbnail:: ../_static/tenants/image118.png
		:width: 300
		:align: center

\#. Нажмите 'Добавить'.

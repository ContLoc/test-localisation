Лицензирование
==============

.. внимание::

	Страница с лицензией на платформу доступна только пользователям с ролью владельца системы.

Платформа поставляется со встроенной лицензией на 30 дней, рассчитанной на 128 ядер ЦП. Данная лицензия может использоваться для пилотных проектов в средах разработки и тестирования.

Чтобы открыть страницу с лицензией платформы, выберите «Системные настройки» -> «Лицензия» на левой панели.

.. thumbnail:: ../\_static/media/goto\_liecense.png

Давайте рассмотрим информацию на этой странице.

.. thumbnail:: ../\_static/media/current\_license.png

1. «Текущая лицензия» — информация о действующей лицензии пользователя.
2. «Модули» — информация о лицензиях на дополнительные модули платформы.
3. «ИНСТРУКЦИИ ПО ПРОДЛЕНИЮ ЛИЦЕНЗИИ» — для продления лицензии следуйте инструкциям на этой странице.

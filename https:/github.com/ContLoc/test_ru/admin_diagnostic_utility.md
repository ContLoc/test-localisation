Диагностический инструмент
===============================

Начиная с версии 2.1, в установочный пакет включен диагностический инструмент sdc-tantor.

Он должен использоваться в случае сбоя при установке или обновлении платформы.


.. важно::

   Если в атрибутах файла нет флага выполнения, установите его с помощью следующей команды:

   .. блок кода:: bash

      # chmod 755 ./sdc-tantor

Команда для запуска утилиты:

.. блок кода:: bash

   \# ./sdc-tantor

Результат ввода команды:

   ::

      SDC-Tantor (версия 1.0.0)
      Будут собираться данные о конфигурации системы и диагностическая информация.
      из этой ОС.
      ...

Сообщение в консоли о завершении диагностической процедуры:

   ::

      Сбор данных завершен.
      Создание архива...
      Архив создан и сохранен в:
            sdc-report_test-01.tantorlabs.ru_20230718_194100.tar.gz
      Пожалуйста, приложите этот файл к заявке в службу поддержки.
      #

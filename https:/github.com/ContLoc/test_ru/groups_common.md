Управление группами пользователей
=================================

В этом разделе описаны действия с группами пользователей платформы.

Чтобы перейти на страницу управления группами пользователей, нажмите «Конфигурация клиента» → «Группы»:

.. thumbnail:: ../\_static/media/config\_menu\_groups.png

Откроется страница управления группами пользователей:

.. thumbnail:: ../\_static/media/user\_groups\_common\_1.png

1. Поле поиска для поиска группы по названию.
2. Кнопка обновления страницы.
3. Группы можно фильтровать по типу: локальные или внешние.
4. Кнопка «Добавить внешнюю группу» открывает страницу :ref:`«Внешняя аутентификация» <ldap>` .
5. Кнопка «Добавить локальную группу» открывает окно для добавления новой группы.
6. Значок шестеренки, с помощью которого можно фиксировать, удалять или изменять положение столбцов в таблице.

	.. thumbnail:: ../\_static/media/group\_table\_setting.png :width: 300 :align: center

7. Информация о группах:

	* название группы;
	* описание группы;
	* Тип группы: локальная или внешняя;
	* количество пользователей, добавленных в группу;
	* количество рабочих пространств, к которым имеют доступ пользователи в группе.

8. Меню «Группа» — позволяет удалить группу. В появившемся окне введите «удалить» и нажмите «Удалить группу».

	.. thumbnail:: ../\_static/media/user\_groups\_common\_1.png :width: 300 :align: center

.. примечание::

	Для фильтрации информации в таблице щелкните заголовок столбца.
